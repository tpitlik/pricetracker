О проекте

Основная цель данного проекта - предоставить всем желающим удобный ненавязчивый инструмент для отслеживания изменения цен на товары в каталоге портала onliner.by

С помощью сервиса вы сможете:

  • Создать собственный список товаров для отслеживания цен на них
  • Проанализировать динамику цен на выбранные товары
  • Своевременно принять решение о покупке товара
